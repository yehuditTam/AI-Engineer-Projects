# NL2CMD - Natural Language to CMD Agent

Agent שמקבל טקסט בעברית וממיר לפקודת Windows CMD.

## 🎯 מטרת הפרויקט

ללמוד **Prompt Engineering** דרך תהליך איטרטיבי:
1. נסח פרומפט
2. בדוק עם תרחישים
3. מדוד ביצועים
4. זהה בעיות
5. שפר → חזור לשלב 2

## 🚀 התקנה והרצה

### 1. התקנה
```bash
cd nl2cmd
pip install openai python-dotenv gradio
```

### 2. הגדרת API Key
```bash
copy .env.example .env
# ערוך .env והוסף את המפתח שלך
```

### 3. הרצה

**בדיקות בסיסיות (19 תרחישים)**:
```bash
cd src
python run_tests.py
```

**בדיקות מאתגרות (21 תרחישים)**:
```bash
python run_challenging_tests.py
```

**הערכה עם מדדים**:
```bash
python run_evaluation.py
```

**ממשק Gradio**:
```bash
python app.py
```

## 🎓 מה למדנו

### עקרונות Prompt Engineering
1. **התחל פשוט** - פרומפט בסיסי קודם
2. **בדוק לעומק** - כולל edge cases
3. **מדוד מדויק** - מדדים מרובים
4. **שפר ממוקד** - בעיה אחת בכל פעם
5. **תעד הכל** - למידה מכל איטרציה

### בעיות שזוהו ותוקנו
1. ✓ פרמטרים מיותרים → הוספת כלל מפורש
2. ✓ הוראות דו-משמעיות → סירוב עם ERROR
3. ✓ משימות מרובות → רק הפעולה הראשונה
4. ✓ פקודות מסוכנות → דרישה לפרטים מדויקים

## 📊 מדדי הערכה
