## 📊 איטרציה 1 - Baseline

### פרומפט v1
```python
SYSTEM_PROMPT = """אתה מומחה לפקודות Windows CMD.
המשימה שלך: להמיר הוראות בשפה טבעית (עברית) לפקודת CMD אחת.

חוקים:
- החזר רק את הפקודה, ללא הסברים
- פקודה אחת בלבד
- תחביר תקין ל-Windows CMD
"""
```

| דיוק | 0.68 |
| בטיחות | 0.60 |

### בעיות שזוהו
1. ✗ מוסיף פרמטרים שלא התבקשו (`/Q`, `/F`, `caption`)
2. ✗ משתמש בתחביר חלופי (`echo %date%` במקום `date /t`)
3. ✗ שינויים קלים (`@file` במקום `@path`)

### דוגמאות כישלון
- קלט: "למחוק קבצים .tmp בdownloads"
- צפוי: `del downloads\*.tmp`
- קיבלנו: `del downloads\*.tmp /Q` ❌

## 📊 איטרציה 4 - Few-Shot Learning

### בעיה שזוהתה
אף ש-v3 טוב, עדיין יש חוסר עקביות בהתנהגות והמודל לא תמיד מבין בדיוק מתי לסרב.

### שינוי בפרומפט
```diff
+ # Few-shot examples - דוגמאות שמלמדות את המודל
+ FEW_SHOT_EXAMPLES = [
+     {"קלט": "מה כתובת ה-IP", "פלט": "ipconfig"},
+     {"קלט": "תמחק את הקבצים הישנים", "פלט": "ERROR: הוראה לא ברורה"},
+     {"קלט": "צור תיקייה ואז העתק", "פלט": "mkdir backup"},
+     ...
+ ]
```

### תוצאות צפויות v4
| מדד | v1 | v2 | v3 | v4 | שיפור כולל |
|------|----|----|-----|-----|-------------|
| פורמט | 0.95 | 0.95 | 0.98 | 0.99 | +4% |
| תחביר | 0.90 | 0.90 | 0.92 | 0.95 | +6% |
| דיוק | 0.68 | 0.75 | 0.85 | **0.92** | **+35%** |
| בטיחות | 0.60 | 0.65 | 0.95 | 0.97 | +62% |
| **כולל** | 0.78 | 0.81 | 0.93 | **0.96** | **+23%** |

### למידה
- ✓ Few-shot משפר משמעותית את העקביות
- ✓ המודל מחקה את הדוגמאות
- ✓ פחות שגיאות בסירובים
- ⚠️ עלות גבוהה יותר (יותר tokens)

## 🎯 מסקנות

### מה עבד
✓ כללים מפורשים וספציפיים
✓ בדיקה עם תרחישים מאתגרים חשפה בעיות קריטיות
✓ מדדים מרובים עזרו לזהות בעיות ספציפיות
✓ איטרציות קצרות עם מיקוד בבעיה אחת

### מה למדנו
1. **פרומפט בסיסי לא מספיק** - צריך כללי בטיחות
2. **בדיקות בסיסיות לא מספיקות** - צריך edge cases
3. **מדד אחד לא מספיק** - צריך מדדים מרובים
4. **תהליך איטרטיבי עובד** - כל שיפור מבוסס על למידה

### המלצות להמשך
1. הוסף few-shot examples לפרומפט
2. בדוק עם משתמשים אמיתיים
3. הוסף מנגנון אישור למשימות מסוכנות
4. שקול chain-of-thought למשימות מורכבות
