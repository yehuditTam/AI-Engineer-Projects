# אפליקציית מנהל המשימות החכם

## הפעלה

### 1. הפעל את השרת (אם הוא לא רץ):
```bash
python main.py
```

### 2. פתח את האפליקציה:
פשוט פתחי את הקובץ `client/index.html` בדפדפן (לחיצה כפולה).

### 3. התחילי לצ'אטט!
כתבי בטקסט חופשי:
- "תוסיפי לי משימה: לקנות חלב, קוד 200, עד מחר"
- "מה המשימות שלי?"
- "עדכני משימה 200 לסטטוס בוצע"
- "תמחקי משימה 200"

זהו! 🎉
